# 要件定義書：競合分析レポート自動生成ツール

**作成日:** 2026-02-14  
**バージョン:** 1.0  
**プロジェクト名:** Competitive Insight Automation (CIA)

---

## 1. プロジェクト概要

### 1.1 目的
マーケター向けに、競合企業のWebサイト・SNSを自動分析し、週次レポートを配信するSaaSツールを開発する。

### 1.2 ビジネスモデル
- **価格帯:** ¥14,800~/月（サブスクリプション）
- **ターゲット:** 中小企業のマーケティング担当者、マーケティングエージェンシー
- **価値提供:** 手動で行う競合調査の時間を90%削減

---

## 2. 機能要件

### 2.1 コア機能

#### F1: 競合URL登録機能
- **F1.1** ユーザーが競合企業のURL（Webサイト、Twitter、Facebook、Instagram等）を登録できる
- **F1.2** URLの妥当性チェック（アクセス可能性、形式検証）
- **F1.3** 1アカウントあたり最大10社まで登録可能
- **F1.4** 競合企業に任意の名前（ニックネーム）を付与できる

#### F2: 自動データ収集機能
- **F2.1** Webサイトの変更検知（新規ページ、コンテンツ更新、削除）
- **F2.2** SNS投稿の収集（Twitter/X、Facebook、Instagram）
- **F2.3** メタデータ収集（タイトル、description、OGP情報）
- **F2.4** 週次で自動実行（日曜日23:00に実行）
- **F2.5** 画像・動画の変更検知
- **F2.6** キーワード出現頻度の分析

#### F3: 分析機能
- **F3.1** コンテンツ更新頻度の分析
- **F3.2** SNSエンゲージメント分析（いいね、シェア、コメント数）
- **F3.3** トピック抽出（主要キーワード、トレンド）
- **F3.4** 競合比較（複数企業の活動量比較）
- **F3.5** 前週比の変化レート算出

#### F4: レポート生成機能
- **F4.1** 週次レポートの自動生成（PDFまたはHTML）
- **F4.2** レポート内容：
  - エグゼクティブサマリー
  - 競合ごとの活動サマリー
  - 注目すべき変更点のハイライト
  - SNSエンゲージメント推移グラフ
  - トレンドキーワードランキング
- **F4.3** カスタマイズ可能なレポートテンプレート
- **F4.4** 日本語・英語対応

#### F5: 配信機能
- **F5.1** メール配信（毎週月曜日9:00）
- **F5.2** Slack連携（Webhook）
- **F5.3** ダッシュボードでの閲覧
- **F5.4** 過去レポートのアーカイブ（最大52週分）

#### F6: ダッシュボード機能
- **F6.1** ユーザー登録・ログイン（OAuth対応）
- **F6.2** 競合企業管理画面
- **F6.3** レポート閲覧画面
- **F6.4** 設定画面（配信時刻、通知方法）
- **F6.5** 使用状況ダッシュボード

### 2.2 管理機能

#### F7: ユーザー管理
- **F7.1** アカウント作成・削除
- **F7.2** プラン管理（Free/Basic/Pro）
- **F7.3** 請求管理（Stripe連携）
- **F7.4** 利用統計の表示

## 4. 技術要件

### 4.1 技術スタック候補

- **データベース:** PostgreSQL 15+ (リレーショナル), Redis (キャッシュ)

- **UI:** Tailwind CSS, shadcn/ui

#### インフラ
- **ホスティング:** AWS / GCP / Vercel
- **コンテナ:** Docker + Docker Compose
- **CI/CD:** GitHub Actions
- **モニタリング:** Sentry, CloudWatch / GCP Logging

#### 外部サービス
- **スクレイピング:** Playwright / Puppeteer

- **メール配信:** SendGrid / AWS SES

### 4.2 開発環境
- **バージョン管理:** Git (GitHub)
- **コードフォーマット:** Black / Prettier
- **リンター:** Ruff / ESLint
- **テスト:** pytest / Jest

## 5. ユーザーストーリー

### US1: 初回登録
**As a** マーケティング担当者  
**I want to** 競合企業のURLを簡単に登録できる  
**So that** 手動での調査作業を自動化できる

**受け入れ基準:**
- URLフォームにペーストして「追加」ボタンをクリック
- エラーがあれば即座にフィードバック
- 登録後、確認画面が表示される

### US2: 週次レポート受信
**As a** マーケター  
**I want to** 毎週月曜日に競合の動向レポートを受け取る  
**So that** 週の戦略立案に活用できる

**受け入れ基準:**
- 月曜9:00にメールが届く
- レポートは視覚的でわかりやすい
- スマホでも読みやすい

### US3: 異常検知通知
**As a** マーケター  
**I want to** 競合が大きな変更を行った際に即座に通知を受け取る  
**So that** 迅速に対応できる

**受け入れ基準:**
- 新規ページ公開、大幅なサイトリニューアル時に通知
- Slackまたはメールで受信
- 通知頻度の設定が可能

## 6. 制約事項

### 6.1 技術的制約
- **C1.1** 一部SNSはAPI制限があるため、公開情報のみ収集
- **C1.2** スクレイピングはrobot.txtを遵守
- **C1.3** レート制限により、大量のURLを短時間で処理できない

### 6.2 ビジネス制約
- **C2.1** 初期開発期間：3ヶ月以内
- **C2.2** 初期開発予算：想定せず（個人開発または小規模チーム）
- **C2.3** MVP（最小機能プロダクト）を優先

### 6.3 法的制約
- **C3.1** スクレイピングは利用規約に違反しない範囲で実施
- **C3.2** 著作権・知的財産権の尊重
- **C3.3** 個人情報保護法・GDPR対応
